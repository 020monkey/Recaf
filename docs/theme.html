Summary

JavaFX allows components to be styled using CSS. Recaf comes with a few stylesheets (Some are incomplete, flat is the default and is complete) which can be specified on the options panel under Display. The value in the option field specifies the suffix to use for files located in src\resources\style. If the file does not exist, the flat theme is used as a default.

To create your own style, copy the flat theme CSS and edit to your liking. Tags are sorted into groups and have some comments explaining choices made for the styles and where they are used in the UI.
