Recaf

What is Recaf?

Recaf is a Java 8 bytecode editor based on Objectweb's ASM. ASM is a library that abstracts away the constant pool and class-file attributes. Without having to worry about satisfying the rigid class-file structure editing bytecode becomes a breeze. With additional features to assist in the process of editing Recaf blows competition out of the water.
