What is Recaf?

Recaf is an open-source Java bytecode editor based on Objectweb's ASM. ASM is a library that abstracts away the constant pool and class-file attributes. Since keeping track of the constant pool (and also stackframes. ASM will regenerate those for you) is no longer necessary complex changes can be made with relative ease. With additional features to assist in the process of editing Recaf is the most feature rich free bytecode editor available.

Recaf requires Java 8 to run and the UI is made with JavaFX. These can be satisified by installing any JRE/JDK beyond 8. Some more advanced features will require running under a JDK (such as attaching to running processes and recompiling code from decompiled classes.)

Recommended information

While ASM makes everything very simple it does not mean you can dive head-first into editing compiled java programs without understanding the following:

  • Basic programming concepts
    • Knowing how control-flow (if / while / for) work under-the-hood.
    • General feel for how to design a basic program (Intuitive feel for identifying logic in potentially obfuscated filest)
  • Java's class file design
